## 1️⃣ Customer Churn Analysis Dashboard (Power BI)

**Objective:**  
Analyze customer churn behavior and identify key factors influencing customer attrition.

**Key Highlights**

- Built interactive **Power BI churn analytics dashboard**
- Performed customer segmentation using **tenure, contract type, and service usage**
- Created KPI measures using **DAX**
- Identified churn drivers and retention opportunities

**Key Insights**

## 4️⃣ Marketing Campaign Performance Dashboard (Power BI)

**Objective:**  
Evaluate marketing campaign effectiveness and customer spending behavior.

**Key Highlights**

- Built a **Power BI dashboard for marketing analytics**
- Performed **customer segmentation based on income and spending**
- Evaluated **campaign response rates**

**Key Insights**

- Majority of customers fall in the **40–60 age group**

- Power BI  
- Power Query  
- DAX

# Superstore Sales Performance Dashboard

## Project Overview
This project analyzes retail sales performance using Tableau. The dashboard provides insights into regional sales performance, profitability, product category contribution, and discount impact.

---

# TDSP Methodology

## 1. Business Understanding

Retail organizations require continuous monitoring of sales performance and profitability.

Key questions addressed:
- Which regions generate the highest sales?
- Which product categories are most profitable?
- How do discounts affect profitability?
- What are the sales trends over time?

---

## 2. Data Understanding

Dataset: **Superstore Sales Dataset**

Main variables include:

| Feature | Description |
|------|------|
Sales | Revenue generated |
Profit | Profit earned |
Discount | Discount applied |
Region | Sales region |
Category | Product category |
Order Date | Order transaction date |

---

## 3. Data Preparation

Steps performed:

- Data validation
- Data type correction
- Creating profit margin metrics
- Aggregating sales trends by month

---

## 4. Modeling

Calculated metrics created in Tableau:

Profit Margin : SUM([Profit]) / SUM([Sales])


---

## 5. Visualization

Dashboard Components:

KPI Section
- Total Sales
- Total Profit
- Profit Margin

Business Insights
- Sales by Region
- Profit by Category
- Profit vs Discount

Time Analysis
- Monthly Sales Trend

---

## 6. Key Insights

- West region generates the highest revenue.
- Technology category contributes the highest profit.
- High discounts significantly reduce profitability.
- Sales show consistent growth across the analyzed period.
